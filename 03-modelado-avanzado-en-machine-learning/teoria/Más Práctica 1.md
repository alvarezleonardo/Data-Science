Más Práctica 1

Ejercicio

Tomar el dataset de **Bikes** (habilitado para descarga), evaluarlo y hacer un
análisis exploratorio.

Realizar la **ingeniería de features** correspondiente (si se necesitaran variables
dummies, considerarlo). Ver si se pueden crear las siguientes features:

- **hora:** como una sola feature numérica (de 0 a 23).
- **hora:** como una feature categórica (usar 23 variables dummy).
- **día:** como una sola feature categórica (`día = 1` de 7am a 8pm, `día = 0` en
  caso contrario).

Crear el modelo de **regresión** correspondiente y evaluar posibles modelos con
**variables cuadráticas**, probando si funcionan mejor.
