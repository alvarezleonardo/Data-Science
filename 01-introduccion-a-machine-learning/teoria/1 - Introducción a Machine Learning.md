Inteligencia Artiﬁcial vs. Machine Learning

Inteligencia Artiﬁcial (IA) y Machine Learning (ML) son dos términos muy populares hoy en día. A menudo se
usan como si signiﬁcaran lo mismo, pero en realidad no son exactamente iguales.

Inteligencia Artiﬁcial

➔ Es la capacidad de una computadora o máquina para imitar el
comportamiento inteligente humano y realizar tareas que
normalmente haría una persona.

➔ Es un campo amplio que incluye muchas técnicas para crear
máquinas que pueden realizar tareas similares a las humanas.

Machine Learning

➔ Se enfoca en una cosa particular: cómo las máquinas pueden

aprender de los datos o experiencias pasadas.

➔ Es una parte de la IA que se centra en cómo las máquinas pueden

aprender de los datos por sí mismas, sin ser programadas con reglas
especíﬁcas.
